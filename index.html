Introduction

HTML (Hyper Text Markup Language), is part of the "holy trinity" of front-end web development, together with CSS and Javascript.


It's not properly a programming language, but instead a markup language, that is, it displays the elements accordingly to the tags they are in.


The broswer interprets the code and displays it in a readable manner to the user, displaying text, input fields, images, buttons etc.


It's current version is HTML5, which is standarized by the W3C (World Wide Web Consortium), an international organization responsible of stablishing coherent patterns to the internet.

Basic Structure

The basic structure of an HTML page firstly consists of some basic structure tags, so that the browser might correctly identify and interpret the document.


The DOCTYPE tag must always be the first to appear in an HTML document, as it identifies the current version being used.


Secondly comes the html tag, which determines the limit of document, which means all other tags must be within it.


Next comes the head tag. The content of this tag is not visible on the browser, which shows the body of the document instead.

It instead, defines aspects and isntructions pertainig content and behaviour.


The meta tag, which goes inside the head tag, is maninly used to specify the characters group to be used in the document, the most used being UTF-8.


The title tag defines the name of the page, which will be displayed at the browser's tab.


Finally we have the body tag, it marks the space of the document that will actually display content in the browser. All tags that contain visual elemnts should be between this tag.


Below follows an exmaple showing all mentioned tags together to form a basic structure:


<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8"/>
    <title>Document</title>
</head>
<body>

</body>
</html>

Headers

Headers are mostly used for title of pages, or sections of the page. It can also function as a way to pinpoint a "landmark" on your document content. It starts from h1 and goes up to h6. h1 is the largest in size and h6 the smallest.

Here's a code example:


<h1>header 1</h1>
<h2>header 2</h2>
<h3>header 3</h3>
<h4>header 4</h4>
<h5>header 5</h5>
<h6>header 6</h6>

Paragraphs

Text paragraphs are generated in html documents by the use of the p tag. It is the most commonly used tag to write simple text in html. It hasit's display in blocks, which means each p tag will be displayed below the previous one.


Here we can see a snippet of the syntax:


<p>First paragraph.</p>
<p>Second paragraph.</p>
<p>Third paragraph.</p>
<p>Fourth paragraph.</p>

Lists

Lists in html are like lists in any other sort of program or even in real life, it's purpose is to organize items in a concise fashion. The two mainlists in html are ul (unordered list) and ol (ordered list). inside the mentioned list tags, there must li (list item) tags, which defines each item to be displayed in the list.


Below are examples of syntax and what they actually print in the document:


<ul>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ul>
  • Item 1
  • Item 2
  • Item 3

<ol>
    <li>Item 1</li>
    <li>Item 2</li>
    <li>Item 3</li>
</ol>
  1. Item 1
  2. Item 2
  3. Item 3

Images

Images in an html document are made trought the img tag. It's a self closing tag not needing a second tag with "/". In the src (source) attribute, the archive address is indicated, it is needed so the document may fetch the file to display it. It also has another important atribute which is alt, it displays the inserted text if the image is unavailable or cannot be reached.


Here's a code example:


<img src="source_image.jpg" alt="placeholder text"/>
